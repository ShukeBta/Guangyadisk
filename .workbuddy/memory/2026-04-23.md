## 光鸭云盘插件 - 关键修复：统一key/目录/类名（18:27）
### 根本原因分析
通过深入分析日志和MoviePilot V2插件规范，找到多个根本问题：

1. **`package.v2.json` 格式错误**：
   - 之前用 `guangyapan` 作为 key → 应该是**类名** `GuangYaDisk`
   - icon 用了完整 URL → 应该是**文件名**

   - 之前：目录名 `guangyapan` ≠ 类名 `GuangYaDisk`
   - MoviePilot 通过 package.v2.json 的 key 找 `plugins.v2/{key}/` 目录
   - 目录名必须与 key（即类名）一致

   - `'NoneType' object has no attribute 'strip'` → /api/v1/plugin/remotes 解析package.v2.json时字段不匹配
   - `{"detail":"Not Found"}` → Vue前端路由找不到（因为插件未正确加载）
   - 插件日志404 → 插件从未成功初始化
